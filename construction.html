Phonology: The Three-Tier System

Joylang uses a three-tier architecture: everyday core sounds, universally accessible consonants, and an advanced set reserved for experimental use.

Tier 1 — Core Vowels

These five vowels exist in almost every major language — Hindi, Spanish, Italian, Japanese, Swahili, Tamil. A speaker of any of these has zero learning cost.

Tier 3 — Naviku Varnu (Advanced Sounds)

Naviku Varnu — navi (new/advanced) + varnu (sound/letter) — is the extended sound inventory for Joylang. These 36 sounds are optional and never required for standard communication. Native speakers whose languages already contain these sounds may use them naturally; Joylang accepts them without penalty. Write Tier 3 sounds inside angle brackets ⟨ ⟩ in formal notation to signal their optional status.

When to use Naviku Varnu Standard speech & writing: never mandatory. Experimental dialect "Joylang-Deva": all Tier 3 sounds active. Loanwords & proper nouns: use to preserve authenticity (e.g., keeping "ق" in "Qatar", "ñ" in "España").

Phonotactics — Word Shape Rules

Chosen Structure: CVCV with Controlled Extensions. Every root word tries to follow the shape CV·CV (Consonant-Vowel repeated).

Basic Rule — The CVCV Pattern

mala → m-a-l-a ✓
kito → k-i-t-o ✓
venu → v-e-n-u ✓
shabi → sh-a-b-i ✓

Extended Rule — CVC Endings

When grammar suffixes are added, a word may end in a consonant. A word may end in a consonant only if that consonant is: n, m, s, l, r — the softest, most universally pronounceable final consonants.

Syllable Structure Summary

PatternAllowed?Example
CV✓ Yesma, ki, to
CVC (final only, soft consonants)✓ Yesman, kil
CVCV✓ Yes (standard)mala, kito
CVCVC (with suffix)✓ Yesmalachi
VCV (vowel-start)✓ Yesanu, ema
CCV (limited stop+liquid only)✓ Limitedpranu, klibo, trenu
CCVC✗ No — Tier 3 only—
⚠ Common Mistake Never end a word with: p, b, t, d, k, g, f, v, h, sh, ch, j. Words may only end in vowels (a e i o u) or the soft consonants n m s l r.

Stress & Rhythm — Syllable-Timed + Penultimate Stress

Rhythm type: Syllable-timed — every syllable takes roughly equal time (like Spanish, French, Swahili).

Stress Rule: Stress always falls on the second-to-last syllable (penultimate). No exceptions.

WordStress Pattern
malaMA-la
kitomeki-TO-me
venushabive-nu-SHA-bi
kachiloka-CHI-lo
kachishiloka-chi-SHI-lo
mavombuma-VOM-bu
Why Penultimate? It is the most common stress pattern on Earth (Latin, Italian, Polish, Swahili, Indonesian). It is perfectly predictable — no exceptions. It gives the language a pleasant, bouncy rhythm. Stress never changes a word's meaning.

Joylang is pronunciation-tolerant. No speaker is corrected for natural accent variation within Tier 2 sounds.

Grammar: Lightly Agglutinative

Meaning is built by sticking suffixes to root words in a predictable order. Structure:

Core Grammar Formula ROOT + [object-marker] + [tense-marker] + [plural-marker]
Every suffix has exactly one job. No suffix blends two meanings together.

Word Order: SOV with Flexibility

Full sentence structure: [Subject] [Adverb] [Object] [Verb]

Mika helo vanu kita.
Mika quickly rice eat. → "Mika quickly eats rice."

Flexibility Rule: Because this language uses case markers (suffixes that show who is subject, who is object), the word order can be changed for emphasis without confusion:

Vanu Mika kita.
→ It is RICE that Mika eats. (object fronted)
Kita Mika vanu.
→ EATING is what Mika does to rice. (verb fronted — poetic)
Standard order is SOV. Deviation is allowed for emphasis.

Writing System: The Contradiction Resolved

An early design decision had an apparent contradiction:

  • Chosen: Historical spelling — spelling preserves history
  • But also stated: No silent letters, One sound = one spelling

Resolution: Historical spelling is dropped. The language uses perfectly phonetic spelling, matching the stated design constraints.

New Spelling Rule What is written is exactly what is spoken. No exceptions. No silent letters. No irregular spellings.

Devanagari Script

Natural fit for Indian-origin primary speakers. Already phonetically organized (each character = one sound). Widely available on keyboards and devices across South Asia.

Future Own Script — Design Principles

When the own script is developed, it must follow these principles:

  • One glyph = one sound. No ambiguity.
  • Vowels are full letters, not marks added to consonants. (Unlike Devanagari's vowel marks.)
  • Direction: Left-to-right (matches global digital standard).
  • Aesthetic: Rounded, friendly shapes — matching the fun and warm personality of the language.
  • Legibility: Each glyph distinct enough that even messy handwriting is readable.
  • Tier marking: A built-in way to mark Tier 3 sounds without extra notation.
