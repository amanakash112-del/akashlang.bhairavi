The Bhairavi Script

Joylang's own writing system — original, pen-friendly, one glyph per sound.

v1.0Final selection
What this is Bhairavi is Joylang's own alphabet — not derived from Latin, Devanagari, or any existing script. Every sound has exactly one glyph, vowels are full letters (never marks), it runs left-to-right, and Tier 3 sounds are formed by adding a red diacritic to a base glyph.

Tier 3 by diacritic

A base glyph plus one red mark yields an advanced sound: two dots above = aspiration, a dot below = retroflex, a wave below = voiced, a tilde through = friction, an open ring below = uvular/pharyngeal, an arc above = palatal, a bottom tie = labial-velar, a trailing ! = click, a small ring = rare vowel, and an accent over a vowel = tone. Writing words joined

Bhairavi words can be written three ways, in increasing fluidity. Separate keeps letters distinct (clearest for learners). Headstroke hangs them from one continuous top bar — the way Devanagari binds a word. Flow runs a wavy baseline beneath for a faster, cursive hand. All three read left-to-right with one glyph per sound.
