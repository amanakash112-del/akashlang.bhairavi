Introduction

What Is Joylang?

Joylang is a constructed auxiliary language (conlang) — an invented language designed so that speakers of very different native tongues can communicate easily, warmly, and playfully. It is not a replacement for your native language; it is a bridge.

Imagine a group of friends from India, Japan, Spain, Nigeria, Turkey, and Brazil. Each has a rich language. Each has spent years mastering it. But when they meet, none of them has a shared tongue that is equally easy and equally foreign for everyone. Joylang is designed to solve that problem.

It is nobody's native language — which means the playing field is level for everyone. The language is officially classified as a Joylang-Auxlang Hybrid: mostly easy and accessible like a pure auxiliary language, but with a slight cultural warmth and personality that gives friend-groups a sense of ownership.

Design Motto "Simplicity. Playfulness. Universal Accessibility." — Every decision in Joylang's design is made against these three values.

Three Core Values

ValueWhat It Means in Practice
SimplicityNo irregular verbs. No grammatical gender. No tones. No exceptions. Every rule applies 100% of the time.
PlayfulnessWords are chosen to sound a little delightful. Grammar is gentle, not military. Making mistakes is encouraged.
Universal AccessibilitySounds exist in Hindi, Arabic, Japanese, Spanish, Swahili, Tamil, Turkish, English, Mandarin, and Yoruba. Everyone can pronounce every word.

Formality Registers

Joylang uses two formality registers: one casual (for friends) and one polite (for elders/strangers). Like Hindi's tum/aap distinction. The difference is expressed through pronoun choice and optional vocabulary selection — not through complex grammatical changes.

Writing System

Joylang uses a perfectly phonetic spelling system: what is written is exactly what is spoken. No silent letters. No exceptions. It currently uses two scripts:

  • Roman alphabet — primary international script, no diacritics needed
  • Devanagari — alternative script, natural for Indian-origin speakers
  • Own script — planned for the future with rounded, friendly glyphs

Morphology Overview

Morphological Types

TypeHow It WorksLanguagesEase
Isolating / AnalyticWords don't change. Grammar shown by word order and helper words.Mandarin, VietnameseVery easy
Lightly Agglutinative ✓Stable roots with simple, stackable suffixes. One suffix = one meaning.Turkish, Finnish, TamilEasy to medium
Fusional (Synthetic)Endings fuse multiple meanings into one.Latin, Russian, Hindi (partly)Medium to hard
PolysyntheticEntire sentences packed into one complex word.Inuit languagesVery hard

Joylang chose Lightly Agglutinative: meaning is built by sticking predictable suffixes to stable root words. Each suffix has exactly one job. No suffix blends two meanings together.

Shade Modifiers lumi = light shade  ·  daki = dark shade
Examples: lumi lala = light red (pink)  ·  daki bolu = dark blue (navy)

Evidentiality

Joylang has optional evidential particles that indicate the speaker's source of knowledge. They attach at the end of the clause before any question particle. Omitting the particle makes no claim about the source.

ParticleMeaning
diruI saw it directly / personal observation
shoruI heard it / hearsay
niruI infer it / logical deduction
boruI read it / from a source or text

Politeness System

Joylang has a two-level pronoun distinction: one informal pronoun set for friends and one formal set for elders and strangers. This is similar to the tum/aap distinction in Hindi or tu/vous in French.
