Sentence Translator

Translate any English sentence into Joylang using Gemini AI (free) or Claude. Configure your API key once in the Setup tab, or use offline Rule-Based mode — no key needed.

⚙ API Key Required

This tool needs an AI API key. Google Gemini is free — please add your key in the Setup tab.

AI mode uses Claude to translate with full grammar understanding. Rule-based works offline using dictionary lookup + SOV reordering.
